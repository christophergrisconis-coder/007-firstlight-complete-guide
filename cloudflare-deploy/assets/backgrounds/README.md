# Background Image Slots (Use Licensed Gameplay Screenshots)

Add your own approved 007 First Light screenshots to these file paths.

## Mission backgrounds

Place one image per mission:

- `assets/backgrounds/missions/m01.jpg`
- `assets/backgrounds/missions/m02.jpg`
- ...
- `assets/backgrounds/missions/m18.jpg`

Recommended size: 1920x1080 or larger (16:9).

## Section backgrounds

Optional section-level backgrounds:

- `assets/backgrounds/pages/summary.jpg`
- `assets/backgrounds/pages/missions.jpg`
- `assets/backgrounds/pages/lookup.jpg`
- `assets/backgrounds/pages/achievements.jpg`
- `assets/backgrounds/pages/data.jpg`

## Notes

- Use only images you own or are licensed to use.
- Keep filenames exactly as listed so auto-mapping works without code changes.
- If a file is missing, the app keeps a gradient fallback.
