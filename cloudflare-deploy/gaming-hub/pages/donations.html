Fraud and Safety

  • Do not store raw card data on this site.
  • Use Stripe Checkout or Payment Links so Stripe handles PCI compliance.
  • Enable 3D Secure, Radar rules, and webhook verification for subscription status.

No raw card handling is implemented in this project by design.
